# BLEND
#### BLENDの非公式APIを作ろうとしたが、需要が無いので体温を送信するだけになった

## 必要な物
* Python(Windowsで確認済み)
* BLENDのアカウント(GoogleはNG)

## 使い方
1. 必要なモジュールをインストールします
* ```pip install selenium```
* ```pip install chromedriver_binary```
2. 起動
* ```python start.py```

### 後はコンソールにIDとPWを入力したら自動で体温を送信します
### 体温は36.0固定です
